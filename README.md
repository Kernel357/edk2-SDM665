# EDK2 for SDM665

This is an implementation of EDK2 for SDM665.

## Works on these devices:
- Xiaomi Mi A3 4/64 (laurel_sprout) 
- Xiaomi Redmi Note 8 4/64 (ginkgo)

## Working:
- Display 
- UEFI shell

## Not working:
- USB
